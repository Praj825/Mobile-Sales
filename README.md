# 📱 Mobile Sales Dashboard | Power BI

## 📌 Project Overview
Analyzed **mobile sales data** across top brands and cities in India (2021).

## 🛠️ Tools Used
- **Power BI** (DAX, Power Query)
- **Dataset:** Mobile Sales India 2021 (Kaggle)

## 📊 Key Insights
| # | Insight |
|---|---|
| 1 | Total Sales: **₹769M+**, 19.2K Units across 3,835 transactions |
| 2 | **Xiaomi, Vivo, Samsung, Apple & OnePlus** top brands |
| 3 | **Delhi, Mumbai, Chennai & Ranchi** key performing cities |
| 4 | **UPI** most popular payment method |

## 📈 Dashboard Features
- KPI cards — Total Sales, Units Sold, Transactions
- Brand-wise performance comparison
- City-wise sales breakdown
- Payment method distribution
- Interactive slicers for Brand & City
